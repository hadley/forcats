Drop unused levels

Usage

fct_drop(f)

Arguments

f
A factor

Description

Same as droplevels(), but with consistent name scheme.

Examples

f <- factor(c("a", "b"), levels = c("a", "b", "c"))
f


[1] a b
Levels: a b c



fct_drop(f)


[1] a b
Levels: a b
